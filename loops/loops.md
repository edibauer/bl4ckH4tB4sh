### while
```bash
while some condition; do
  # commands to be executed as long as the condition is true
done

```

### for
```bash
for variable_name in LIST; do
  # commands to be executed for each item in the list
done
```

### case
```bash
case EXPRESSION in
  PATTERN_1)
    # commands to be executed if EXPRESSION matches PATTERN_1
    ;;
  PATTERN_2)
    # commands to be executed if EXPRESSION matches PATTERN_2
    ;;
  *)
    # commands to be executed if EXPRESSION doesn't match any pattern
    ;;
esac
```
